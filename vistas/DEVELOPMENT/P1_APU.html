The economic and financial management of projects (GEFP) is a set of activities and processes that support project development in terms of time and cost control. In order to reinforcement these managerial activities, this application has been developed, which allows users to prepare budgets through the Unit Price Analysis method (APU), considering the following parameters: Personnel work, work in equipment, materials, services, items, administrative expenses.
